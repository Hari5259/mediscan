# 🚀 Overview

MediScan is a modern AI-powered healthcare platform designed to provide smart medical assistance, digital healthcare management, and wellness tracking through an intelligent and scalable full-stack ecosystem.

The platform integrates authentication, AI support systems, health analytics, OCR medicine scanning, emergency support, doctor dashboards, wellness tracking, and modern healthcare utilities into one seamless experience.

# ⚙️ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/Hari5259/mediscan.git
cd mediscan
```

---

## 2️⃣ Install Frontend Dependencies

```bash
npm install
```

---

## 3️⃣ Install Backend Dependencies

```bash
cd backend
npm install
```

---

## 4️⃣ Configure Environment Variables

Create `.env` inside backend folder:

```env
MONGODB_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
PORT=5000
```

---

## 5️⃣ Run Backend

```bash
node server.js
```

---

## 6️⃣ Run Frontend

```bash
npm run dev
```

Visit:
```text
http://localhost:5173
```
